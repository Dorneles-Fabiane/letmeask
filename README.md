   # 🗁 Projeto

   ## ↗ Objetivo

   - Centralizar as perguntas da audiencia durante a realização de eventos ao vivo e/ou online.
   - Permite login sem cadastro, utilizando apenas o e-mail do google (comunicação com a API de login do Google atraves do Firebase).
   - *Atenção! Você precisará estar logado ou logar em uma conta Google.*

   ### 👁 Visão do Usuário

   - Envie perguntas.
   - Vote e eleja as perguntas mais relevantes para serem respondidas.

   ### 👀 Visão do Administrador

   - Destaque as perguntas mais votadas.
   - Marque perguntas como respondidas.
   - Exclua perguntas sem relação com o tema.
   - Encerre a sala após o termino do evento.
